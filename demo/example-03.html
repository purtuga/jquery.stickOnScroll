Example Demo: Option hasStickOnScroll

Example of the use of the setParentOnStick option, which will automatically set the height of the element's parent that will be stuck on scroll.

The yellow navigation bar below will be stuck when page is scrolled, while a the same time it will set the height of its parent and will reset it when the menubar returns to normal page flow. In addition, the boxed scrolling element below also contains the use of this option.
